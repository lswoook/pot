## 프로젝트 배경

What is this for?
* 코로나로 인한 국제적 정서 나누기
* 로컬 정보 공유 :smile:


### Built With
* [Bootstrap](https://getbootstrap.com)
* [JQuery](https://jquery.com)
* [Thymeleaf](https://www.thymeleaf.or)
* RESTful API
* [Info Ip](infoip.io)

### 구현 상황
* 회원가입, 로그인, 로그아웃
* 회원정보 암호화, 회원 권한 관리
* 게시글 작성 & 수정 & 검색

### Todo List

* 해쉬태그 기능 추가로 자주 언급되는 토픽 파악
* UI & UX 개선
